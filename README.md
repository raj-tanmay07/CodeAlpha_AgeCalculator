# CodeAlpha Age Calculator 🧮

A simple **Age Calculator web app** built with **HTML, CSS, and JavaScript**

## 📌 Overview

This web application lets users input their **date of birth** and instantly calculates their age in:

- 🗓 Years  
- 📆 Months  
- 📅 Days  

It’s a beginner-friendly project focused on learning JavaScript date operations and DOM interaction.
